> [!info] About this folder
> **Orphan bucket — only for meetings without a clear project home.**
>
> Meetings about a project belong inside `🚀 Projects/<Project>/Meetings/` (the project folder is the home; the `Meetings/` subfolder keeps things tidy). This folder is the fallback for ad-hoc 1:1s, cross-cutting discussions, or exploratory conversations that haven't crystallized into a project yet.
>
> Use the **Meeting Note** template. The `lark-minutes-debrief` skill auto-routes meetings to the right project folder — only orphan meetings end up here.
>
> **File naming**: `[Topic] - YYYY-MM-DD.md`

> Full convention: see [[CLAUDE#Meeting Notes]]
